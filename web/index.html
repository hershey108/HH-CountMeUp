What is Count Me Up?
This application allows you to vote on candidates, and also track their current vote counts. When voting, please enter your email address and select a candidate. Please note, each user can only vote a maximum of 3 times.
Live updates
The candidates' vote count table is a live update, which refreshes each second. To see this in action please click the following button to have a large number of random votes cast. This will only work once per page load.
